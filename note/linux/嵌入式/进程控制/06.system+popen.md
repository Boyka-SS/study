#### system

```c
#include <stdlib.h>
int system(const char *command);
```

**返回值**

成功，返回进程的状态值；
内部是对 exec 族函数的封装，内部调用了 sh，当 sh 不能执行时，返回 127
失败返回-1

system()函数执行完参数命令会返回调用点继续执行后续代码，而 exec 不会
